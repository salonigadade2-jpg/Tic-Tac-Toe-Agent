This project implements an **AI agent** that plays the classic game of **Tic-Tac-Toe** against a human player. The AI uses game-theory-based algorithms such as **Minimax**, with optional **Alpha-Beta Pruning**, to make optimal decisions and remain unbeatable.

The project provides a hands-on understanding of decision-making algorithms and adversarial search techniques.
